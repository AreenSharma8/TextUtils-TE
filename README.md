# TextUtils - A React Text Utility Application

TextUtils is a simple and powerful text utility tool built with React that helps you modify and analyze your text easily. It provides features like text transformation, live preview, word and character counting, and dark/light mode support for a smooth user experience.

## Features

- **Text Transformation**: Convert text to uppercase or lowercase, and clear content instantly.
- **Live Preview**: See changes in real-time as you type.
- **Word and Character Counter**: Get instant counts of words and characters in your text.
- **Reading Time Estimation**: Calculate the estimated time to read the text.
- **Dark/Light Mode**: Toggle between dark and light themes for comfortable viewing.
- **Responsive Design**: Works seamlessly on desktop and mobile devices.

## Technologies Used

- **React**: Frontend framework for building the user interface.
- **React Router**: For navigation between pages.
- **Bootstrap**: For styling and responsive design.
- **JavaScript**: Core logic and functionality.

## Installation

1. Clone the repository:
   ```
   git clone https://github.com/your-username/textutils.git
   cd textutils
   ```

2. Install dependencies:
   ```
   npm install
   ```

3. Start the development server:
   ```
   npm start
   ```

4. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## Usage

1. Enter your text in the textarea on the home page.
2. Use the buttons to convert to uppercase, lowercase, or clear the text.
3. View the summary including word count, character count, and estimated reading time.
4. Toggle between dark and light modes using the switch in the navbar.
5. Navigate to the About page to learn more about the application.
